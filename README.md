# E-commerce-Customer-Segmentation-using-RFM-Analysis
**End-to-End Customer Analytics Project**  
**Segmenting customers based on purchasing behavior to drive retention and marketing strategies**

---

### 📋 Project Overview
This project analyzes a large e-commerce transactional dataset to segment customers using the **RFM (Recency, Frequency, Monetary)** framework. The objective was to identify high-value customers, loyal buyers, at-risk customers, and dormant users to enable targeted marketing and improve customer lifetime value.

---

### 🎯 Problem Statement
E-commerce businesses generate huge volumes of transaction data but often struggle to understand customer behavior. The goal was to:
- Segment customers based on their purchasing patterns
- Identify high-value and at-risk customer groups
- Provide actionable insights for personalized marketing and retention campaigns

---

### 📊 Dataset
- **Size**: 50,000+ transactions
- **Time Period**: 2010–2011 (UK-based online retail)
- **Key Variables**: InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country
- **Source**: Public e-commerce dataset (anonymized)

---

### 🛠️ Tools & Technologies
- **Python**: Pandas, NumPy
- **Analysis**: RFM Scoring, Customer Segmentation
- **Visualization**: Power BI Dashboard
- **Others**: Data Cleaning, Exploratory Analysis

---

### 🔍 Key Analysis & Insights
- Calculated **Recency, Frequency, and Monetary** values for each customer.
- Assigned RFM scores (1–5 scale) and created a combined RFM Score (out of 555).
- Segmented customers into four meaningful groups:
  - **High Value**
  - **Loyal**
  - **At-Risk**
  - **Dormant**
- Built customer distribution chart showing the proportion of each segment.
- Prepared cleaned RFM dataset for Power BI to support marketing decisions.

---

### 📈 Visualizations & Dashboard
- Customer Segments Distribution Chart (with count and percentage)
- RFM Score distribution
- Interactive **Power BI Dashboard** with:
  - Segment-wise KPIs
  - Customer distribution visuals
  - Filters for deeper analysis

---

### 💡 Business Recommendations
- **High Value Customers**: Reward with loyalty programs and exclusive offers.
- **Loyal Customers**: Focus on upselling and cross-selling opportunities.
- **At-Risk Customers**: Run personalized win-back campaigns and limited-time promotions.
- **Dormant Customers**: Design re-engagement campaigns and feedback surveys to understand reasons for inactivity.
